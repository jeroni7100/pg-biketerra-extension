Extension Options - Biketerra x Pedal Games addon

Note: The WebSocket address is the address of the server (your Pedal Games application) to which the extension will send data. The default address used by Pedal Games is ws://localhost:49998.

Note: The send interval is the time in milliseconds between each data packet sent to the server (Pedal Games).

Note: The inactivity timeout is the time in seconds after which the extension will stop sending data to the server (Pedal Games) if no data has been received from the game (Biketerra).
